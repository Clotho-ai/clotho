# clotho

An ai orchestrator built to visualize agentic driven workflows.

Where is my context? What agent is working on what work? Can I collaborate on agent direction with my team members yet? Working with AI on your own local environement can be great! This project seeks to create a shared space that can be deployed anywhere, streamed among a group of engineers and interacted with.
